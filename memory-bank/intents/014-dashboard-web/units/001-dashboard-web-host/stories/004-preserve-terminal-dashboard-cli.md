# Story: Preserve Terminal Dashboard CLI

## User Story

As an existing specsmd user, I want the terminal dashboard to remain available under a clear command after `dashboard` becomes web-first.

## Acceptance Criteria

- [ ] `dashboard-cli` runs the existing terminal dashboard.
- [ ] Existing terminal dashboard options are preserved.
- [ ] Help text distinguishes web dashboard from terminal dashboard.

## Notes

Compatibility aliases can be considered during implementation, but `dashboard-cli` is the named target.
