# specs.md-website

This is the documentation website for specs.md, built with **Mintlify**.

## Getting Started

1.  Install Mintlify CLI:
    ```bash
    npm i -g mintlify
    ```

2.  Run the development server:
    ```bash
    mintlify dev
    ```

## Structure

-   `mint.json`: Configuration (Navigation, Colors, Branding).
-   `introduction.mdx`: Main content page.
